## 環境/技術
- EJS
- Sass
- JavaScript または jQuery
- Node.js 12.14.1
- 記法：Scss
- CSS設計：FLOCSS（https://github.com/hiloki/flocss)
- ソースコード管理：GitHub

## 起動方法
 
 1. `git clone https://github.com/daikishirakawa/tenkue-app.git`
 1. `cd tenkue-app`
 1. `npm install`
 1. `npm run start`
